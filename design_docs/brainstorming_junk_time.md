# 脑暴：从“防止平推”到“强迫循环” (Forcing the Loop)

## 1. 之前方案的反思 (Recall)
*   **用户反馈**: "通关税" (Exit Visa) 只是增加了劳动量，没有解决体验的“垃圾时间”本质。
*   **深度复盘**: 问题的核心不在于“钱太多没处花”，而在于**“钱太多导致可以无视游戏规则直接平推”**。
    *   只要有 50g，玩家就可以连续点 10 次刷新 (Refresh)，大概率直接刷出紫装通关。
    *   这个过程不需要动脑，不需要做订单，纯粹是“消耗资源跳过游戏”。

## 2. 新方向：限制“钞能力” (Curbing the Whale)

我们需要一种机制，让玩家即使有 100g，也无法简单地通过“暴力刷新”来直接找齐主线道具。我们必须强迫玩家**回到订单循环中去**。

### 方案 A：通胀刷新 (Inflationary Refresh) —— *强烈推荐*
*   **机制**:
    *   **刷新价格递增**: 连续刷新时，价格飞速上涨。
        *   第 1 刷: 5g
        *   第 2 刷: 8g
        *   第 3 刷: 12g
        *   第 4 刷: 18g ...
    *   **重置条件**: 只有**完成一个订单**，刷新价格才会重置回 5g。
*   **体验改变**:
    *   **旧体验**: 有 50g -> 狂点刷新 -> 买紫装 -> 过关。 (耗时 10秒，无脑)
    *   **新体验**:
        1.  我有点钱，刷了两次没出，现在刷新要 12g 了，太贵了。
        2.  我必须**去做一个小订单**来把价格这一波“热度”降下去。
        3.  为了做这个小订单，我得规划背包、合成物品...
        4.  好，订单做完了，价格回落到 5g，我再试着找主线...
    *   **结果**: 哪怕你有 100g，你也得乖乖地**穿插着做订单**，节奏感就被拉回来了。

### 方案 B：库存禁运 (Inventory Embargo)
*   **机制**: 主线任务要求的“紫装”必须是**“新鲜的”**。
    *   例如：主线要求提交一个“刚刚合成出来不满 3 回合的紫装”。
    *   或者：主线道具占用的格子会**封印该格子周围的物品**，导致你很难受。
*   **评价**: 比较复杂，容易产生负面挫败感，不如方案 A 直观。

### 方案 C：限购令 (Purchase Limit)
*   **机制**: 每个回合（或每完成一个订单前）只能购买/刷新有限次数。
*   **评价**: 硬限制（Hard Limit）通常不如软限制（Soft Limit / Cost Scaling）有趣。

## 3. 推荐实施：方案 A (通胀刷新)
这个改动非常小（代码改动极少），但对节奏的影响是巨大的。它直接把脱钩的“订单循环”和“主线循环”重新挂钩了。

**变种补充**:
*   为了防止玩家觉得太卡手，可以设定：**“每回合自动降低一档价格”** 或者 **“时间流逝也会降价”**。
*   但核心必须是：**做订单是降价最快的方式**。
