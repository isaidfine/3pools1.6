# 脑暴：如何打破 50 分钟的平铺直叙？ (Brainstorming Stage Variations)

## 1. 核心问题
当前版本（全机制解锁）的体验像是一条**水平的直线**。
玩家在第 5 分钟（Stage 0）做的操作（刷新、合成、凑紫色），和在第 45 分钟（Stage 4）做的操作几乎完全一样。唯一的区别是数字（金币多了，价格贵了）。

## 2. 解决方案方向：时代特性 (Era Traits)
为了制造“起伏”，我们不应该砍掉机制，而应该**改变机制的权重**。
我们为每一个时代（Stage）赋予一个**【核心规则 (Theme Rule)】**，这个规则会彻底改变玩家在该阶段的最优策略。

以下是三个不同方向的脑暴：

---

### 方向 A：经济周期的更替 (Economic Cycles)
*核心理念：改变资源的价值，迫使玩家调整收支策略。*

*   **Stage 0 (纯净时代) - 规则：【原始积累】**
    *   **特性**: 初始金币少，但 **没有任何消耗**（刷新免费，或者极便宜）。
    *   **体验**: 爽快。让玩家随意 D 牌，快速凑出第一个紫色，建立信心。
*   **Stage 1 (初识价值) - 规则：【通货膨胀】**
    *   **特性**: 所有物品买入价格 **翻倍**，卖出价格不变。
    *   **体验**: 压力骤增。玩家必须极度精打细算，不敢随意 D 牌，更多依赖“合成”来获取高级物品（因为合成相当于免费升级）。
*   **Stage 2 (风险引入) - 规则：【泡沫经济】**
    *   **特性**: 物品卖出价格 **波动极大** (0.5x ~ 3x)，或者回收时有概率暴击。
    *   **体验**: 赌徒心理。玩家开始尝试囤积物品，等待高价时回收，或者利用“以旧换新”去博取更大的收益。
*   **Stage 3 (策略完全体) - 规则：【消费主义】**
    *   **特性**: 必须 **消费金币** 才能提升主线进度（比如每回合自动扣钱，或者是主线任务额外收税）。
    *   **体验**: 紧迫感。不仅仅是凑物品，而是要保证现金流不断裂。

---

### 方向 B：机制的强化与抑制 (Mechanic Shift)
*核心理念：每个时代让某一个机制变得超强（或失效），引导玩家玩不同的流派。*

*   **Stage 0 - 【合成之世】**
    *   **规则**: 合成不需要“同名”，只需要“同品质”即可！（或者合成会有额外奖励）。
    *   **玩法**: 疯狂买垃圾来合成，体验极速消除的快感。
*   **Stage 1 - 【精选之世】**
    *   **规则**: “刷新”功能被强化（必定出高品质），但“合成”被抑制（合成成功率 50% 或费用极高）。
    *   **玩法**: 放弃合成，专注于利用刷新去找特定的好牌。
*   **Stage 2 - 【置换之世】**
    *   **规则**: 免费开放“有的放矢”或“以旧换新”词缀。
    *   **玩法**: 玩弄手牌结构，体验精准控制的乐趣。
*   **Stage 3 - 【混乱之世】**
    *   **规则**: 所有负面词缀（硬化的、波动的）出现率大幅提升。
    *   **玩法**: 在恶劣环境下求生，考验应变能力。

---

### 方向 C：限制挑战 (Constraints)
*核心理念：通过剥夺某些资源（不是机制）来逼迫玩家思考。*

*   **Stage 1**: **空间幽闭** - 背包只有 4 格（正常是 6-7 格）。逼迫极致的库存流转。
*   **Stage 2**: **强迫症** - 只能提交“同种类”的订单。
*   **Stage 3**: **时间紧迫** - 订单没完成会随时间消失（或刷新次数极少）。

## 3. 建议
考虑到我们希望保留“全机制解锁”的爽快感，我倾向于 **方向 B (机制强化与抑制)** 的变种。
不是剥夺机制，而是**让某个机制在某个时代特别好用**。
这样玩家每进入一个新时代，就会感叹：“哇，这个时代合成这么爽？那我要改变打法了！”

你觉得这三个方向里，哪一个更符合你对“变化”的定义？
